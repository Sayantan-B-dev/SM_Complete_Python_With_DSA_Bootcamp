## What This Website Is About

**VisuAlgo (Graph DS module)** is an interactive visualization platform designed to demonstrate how graph algorithms operate internally through animated, step-by-step execution. It focuses on conceptual clarity by showing how data structures evolve during algorithm processing rather than only presenting final outputs.

---

## How to Use

1. **Open the Graph Section**
   Navigate to the graph data structure module.

2. **Choose an Algorithm**
   Select BFS, DFS, Dijkstra, or others.

3. **Create or Load a Graph**

   * Click to add vertices
   * Drag to connect edges
   * Assign weights if needed

4. **Set Parameters**
   Choose starting node and graph type.

5. **Run Visualization**

   * `Play` for automatic execution
   * `Step` for step-by-step control
   * Adjust speed as needed

6. **Observe Execution**
   Track node visits, edge processing, and internal data structures.
